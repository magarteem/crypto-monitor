# Crypto Monitor - Chancey

Криптовалютная лотерея с современным интерфейсом и блокчейн технологией.

## Структура проекта (FSD-inspired)

```
app/
├── _widgets/          # Переиспользуемые виджеты (Header, Sidebars)
│   ├── Header/
│   ├── ProfileSidebar/
│   └── SettingsSidebar/
├── _pages/           # Страницы приложения (зарезервировано для будущего)
├── _shared/          # Общие ресурсы (зарезервировано для будущего)
├── auth/             # Страница авторизации
├── dashboard/        # Приватная страница dashboard
├── CryptoChart.tsx   # Компонент графика криптовалюты
├── layout.tsx        # Корневой layout
├── page.tsx          # Главная страница с графиками криптовалют
└── globals.css       # Глобальные стили
middleware.ts         # Middleware для защиты маршрутов
```

## Технологии

- **Next.js 16** - React фреймворк
- **TypeScript** - Типизация
- **CSS Modules** - Модульные стили
- **Recharts** - Библиотека для интерактивных графиков
- **Binance API** - Реальные данные криптовалют
- **Biome** - Линтер и форматтер

## Функционал

### Авторизация (`/auth`)
- Форма входа/регистрации с валидацией
- Показ/скрытие пароля
- Обработка ошибок
- Cookie-based авторизация
- Защита маршрутов с автоматическим редиректом

### Главная страница (`/`)
- **Интерактивные графики** криптовалют в реальном времени
- **12 отслеживаемых монет**: BTC, ETH, BNB, SOL, XRP, LINK, LTC, ATOM, CHZ, AAVE, INJ, GNS
- **Обновление цен каждые 5 секунд** с Binance API
- **Линия текущей цены** (оранжевая пунктирная)
- **Zoom функционал** - приближение/отдаление графиков (100% - 400%)
- **Hover эффект** с детальной информацией
- **Градиенты** под линиями графиков
- **Цветовая индикация** - зеленый для роста, красный для падения
- **Процент изменения** за 24 часа

### Dashboard (`/dashboard`)
- Header с навигацией
- Hero секция
- Информация о лотерее
- Feature карточки
- Profile Sidebar с информацией о пользователе

### Общие компоненты

#### Header
- Логотип с редиректом на главную (`/`)
- Навигация: Crypto, Dashboard, Earn, Govern, Learn
- Кнопка профиля с email и сетью
- Адаптивный дизайн
- Полная ширина экрана (без ограничений max-width)

#### ProfileSidebar
- Информация о пользователе
- Wallet балансы
- Redeem опции (Free Tickets, Pot Claim)
- Кнопка DISCONNECT с логаутом

#### SettingsSidebar
- Выбор темы (темная/светлая)
- Настройка уведомлений
- Выбор языка (русский/английский)
- Сохранение настроек в Local Storage

## Защита маршрутов

- **Middleware** проверяет наличие cookie `auth_token`
- **Если НЕ залогинен** - редирект на `/auth`
- **Если залогинен и на `/auth`** - редирект на `/`
- **Logout** удаляет cookie и редиректит на `/auth`

## API

Используется **Binance Public API** для получения данных:
- `/api/v3/klines` - исторические данные (96 точек за 24 часа)
- `/api/v3/ticker/24hr` - изменение за 24 часа
- `/api/v3/ticker/price` - текущая цена (обновление каждые 5 секунд)

## Особенности

1. **Реальные данные** - все цены обновляются с Binance API
2. **Интерактивность** - zoom, hover, анимации
3. **Адаптивный дизайн** - работает на всех устройствах
4. **Защищенные маршруты** - middleware контролирует доступ
5. **Оптимизация** - автоматическое обновление только текущих цен
6. **FSD-архитектура** - чистая структура проекта

# PWA Screenshots

Эта папка содержит скриншоты приложения для Rich PWA Install UI.

## Требования:

- **desktop.png**: 1920x1080px (широкий формат, `form_factor: "wide"`)
- **mobile.png**: 390x844px (мобильный формат)

## Как создать скриншоты:

1. Запустите приложение в браузере
2. Откройте DevTools и установите нужный размер viewport
3. Сделайте скриншот основного экрана приложения
4. Сохраните в эту папку с соответствующим именем

Пока не созданы реальные скриншоты, PWA будет работать, но без Rich Install UI.
